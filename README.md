### Install
- `npm i`
- `bower i`
- `gem install slim --no-ri --no-rdoc`

### Run
- dev: `npm run dev`
- prod: `npm run prod`

Open page on http://localhost:8003

### Deploy on heroku
-  heroku buildpacks:set https://github.com/heroku/heroku-buildpack-multi.git
